# Training Blocks

Training blocks group workouts into user-defined phases.

Examples:

- Lean Bulk 2026
- Strength Block
- Bench Focus Block
- Cut Phase
- Hypertrophy Block

## Data Rules

- A workout can optionally belong to one training block.
- New workouts are assigned to the active block if one is selected.
- Deleting a block does not delete workouts. It removes the block assignment.

## Analytics

Block analytics include:

- sessions
- sets
- volume
- PRs
- start/end dates
- current block summary

Future work can add block-vs-block comparisons and best-block leaderboards.
