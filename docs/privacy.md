# Privacy

IronLung Desktop is local-first.

- No account is required for the MVP.
- No analytics tracking is included.
- No cloud upload is included.
- Photos are stored locally by default.
- Body analysis requires explicit user consent per run.
- Data export and import use local JSON.
- Photos and Data & Settings include deleting all local photo data.
- Boostcamp import processes local files and can optionally refresh through a user-controlled local `boostcamp-mcp` helper. IronLung does not store Boostcamp credentials and does not upload imported data.
- IronLung Analyzer stores imported desktop analytics data locally on the phone. Desktop/mobile transfer uses user-controlled JSON files only; no server, account, or cloud sync is added.
- Progress Photo Index is an experimental progress metric. It is not a medical diagnosis, body-fat measurement, or attractiveness rating.

Users should be able to remove local workout data, progress photos, and analysis outputs without contacting a server.
