# Privacy

IronLung Desktop is local-first.

- No account is required for the MVP.
- No analytics tracking is included.
- No cloud upload is included.
- Photos are stored locally by default.
- Body analysis requires explicit user consent per run.
- Data export and import use local JSON.
- Settings include deleting all local photo data.

Users should be able to remove local workout data, progress photos, and analysis outputs without contacting a server.
