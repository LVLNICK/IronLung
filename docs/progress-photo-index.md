# Progress Photo Index

Progress Photo Index is the privacy-safe name for IronLog's local progress-photo metric.

It is:

- experimental
- local-first
- opt-in per analysis
- user-relative
- based on photo quality, pose consistency, capture context, and model limitations

It is not:

- a medical diagnosis
- a body-fat measurement
- an attractiveness rating
- a comparison against other users

## Separate Signals

- Photo Quality Score: exposure, sharpness, framing, contrast, and aspect quality.
- Pose Consistency Score: how comparable the pose/capture setup appears.
- Progress Photo Index: the private 0-100 progress metric.

The UI must always show: "This is an experimental progress metric. It is not a medical diagnosis, body-fat measurement, or attractiveness rating."
